# rooms-occupancy-service

## Versions 
JAVA: OpenJDK 22.0.1
GRADLE: Gradle 8.8

# How to run test from CLI
```
gradle test
```

# How to run app

### Using gradle;
```
gradle bootRun
```

### Using java (great for Docker)
Build jar file :
```
gradle build 
```

or ```gradle bootJar```
```
java -jar build/libs/app-0.0.1-SNAPSHOT.jar
```
